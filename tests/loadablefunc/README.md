# Loadable Function Integration Tests

This directory contains integration tests for TiDB loadable functions (UDF) created via:

```sql
CREATE FUNCTION ... RETURNS ... SONAME 'xxx.so';
```

## Layout

- `run-tests.sh`: build UDF shared objects, start a standalone `tidb-server`, run SQL tests via `mysql_tester`.
- `funcs/`: loadable function source files. Each source file is compiled to a `.so` placed under `lib/`.
- `t/`: test SQL files (`*.test`).
- `r/`: expected results (`*.result`) generated by `run-tests.sh -r <case>`.

## Run

From the TiDB repo root:

```sh
make loadablefunctest
```

Or run the test driver directly:

```sh
cd tests/loadablefunc
./run-tests.sh -s ../../bin/tidb-server
```
